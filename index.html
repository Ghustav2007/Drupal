o que é um CMS?

Bom a cigla CMS significa Content Management System, e é uma ferramente que oferece ferramentas necessarias para publicar e gerir conteudo online. até pessoas que nao tenham conhecimento voltado para deselvolvimento web podem tirar proveito do CMS, e claro, os profissionais podem tirar proveito também

O que é o Drupal?

É Um Sistema de gerenciamento em conteúdo escrito em PHP e visto como uma framework Modular, ele permite criar e gerir conteúdo, modificar a aparencia do site e atomatizar tarefas. O Drupal NÃO precisa de um sistema operacional especifico, porém precisa de um servidor http que seja compativel com PHP, assim como o apache e um banco de dados como mySQL. e o Drupal é um sistema mais flexível que o WordPress, embora tenha mais configurações com as quais você precisará lidar para deixar o site ou blog da maneira como você quer, e para criar um site e blog, se usa estrutura conteúdo aparencia: voltada para para instalações de temas extenções:voltadas para a melhora da funcionalidade do site configurações: como ja diz o nome voltada para as configurações e ajuste de parametros Reportes: OS Relatorios. e ajuda : lugar onde se pode usar para pedir ajuda.

Como instalar o Drupal?

Para instalar o CMS Drupal o caminho mais facil é usar um auto instalador que customa vir na maioria dos serviços de hospedagem. e para instalar manualmente devemos acessar o site oficial do Drupal e baixar o arquivo em formato zip. extrair os arquivos e envia-los para sua conta hospedagem via FTP ou o proprio gerenciador de arquivos do Host
